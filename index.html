    Features

    1. Ghost (Invulnerability)
    2. Heal
    3. Mana infusion
    4. Aqualung (Breathing under water)
    5. Teleport function
    6. Gold coins
    7. Change name

    How to use Dzenaria

    Startup

    Copy Dzenaria.exe to game folder near Terraria.exe. It's works with all version of Terraria(Steam/no Steam).

    Controls

    • F1 — Save location for teleportation
    • F2 — Teleport
    • F3 — Add 100 gold coins
    • F12 — Show promt to change character name
    • Num1 — Heal toggle (First press activate function, second deactivate)
    • Num2 — Mana Infusion toggle (First press activate function, second deactivate)
    • Num3 — Aqualung toggle (Breathing under water, First press activate function, second deactivate)
